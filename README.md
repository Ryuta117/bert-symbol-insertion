# bert-symbol-insertion
## 概要
ペン入力環境での記号入力を補助するため、BERTを用いて丸括弧の自動挿入を行うモデルの実装。

## 背景
ペン入力では括弧などの記号入力が煩雑になりやすい。
本タスクでは、括弧を除去した文章をBERTに入力し、括弧が挿入されるべき位置を予測する。

## 手法
- トークン間のギャップを単位として、開き括弧・閉じ括弧の挿入をバイナリ分類で予測
- 閉じ括弧の予測に開き括弧の確率を特徴量として追加し、括弧の対応関係を考慮
- GapProjectorにより、ギャップ左右のトークン表現を学習可能な重みで加重平均して特徴量を生成

## 使用モデル
- [LINE DistilBERT Japanese](https://huggingface.co/line-corporation/line-distilbert-base-japanese)
